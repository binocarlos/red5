# setup pi

first update the system:

```
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo rpi-update
```

## ffmpeg

```
$ sudo sh pi/bin/installnode.sh
```

## node

```
$ sudo sh pi/bin/installnode.sh
```

# development

These things should be done for a development pi not a live one

## samba

```
$ sudo sh pi/bin/installsamba.sh
```
